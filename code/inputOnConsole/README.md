
### The input is passed by the Scanner (typing on the console).
- 1 - Prints depending on the number's property.

- 2 - Verifies the number of stars of each element. Input example: 

> *--,---,**-

- 3 - Verifies if the moment matches any pattern. Input example: 

> 22:22,12:21,07:30

- 4 - Takes the numbers of wins and ties in the corresponding element and computes. Input example:

> 1,3,2

> 3,1,2
